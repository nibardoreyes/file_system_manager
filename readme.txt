------Script-----

1. Make the script executable: chmod +x script.sh

-----Program------
1. Download command_interpreter.c and save to a directory that is easily accessible.
2. Launch Ubuntu and navigate to the directory where the C file is located.
3. Once inside the directory, execute this code: gcc -o run file_system_manager.c
4. Change permissions if needed to (chmod 777 file_system_manager.c).
5. Lastly, execute this code: ./run
